个人neovim配置，把这个文件夹替换掉默认的配置文件夹即可

On Macos or Linux:

```zsh
# install neo vim
brew install neovim

# check neovim version
nvim --version

# install Nvchad
git clone https://github.com/NvChad/starter ~/.config/nvim && nvim
```

```zsh
cd ~/.config
rm -rf /nvim

git clone https://github.com/blossomi/nvim_config.git
```

- 进入nvim，等待插件安装完成之后运行 `:MasonInstallAll`
